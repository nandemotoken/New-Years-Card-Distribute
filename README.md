# NFT年賀状発行システム

## NFT年賀状とは？  
ブロックチェーン技術のNFTを利用した、電子的に保存できるデジタルカードです。  
『非対面』『非接触』『物の配送なし』で記念に残るカードを取得できます。  

---

### 利用方法
2020/12/31日にブロックチェーン利用手数料の事前登録が必要  
事前登録が実施済みの方は、次の①か②の方法でNFT年賀状を受け取ってください。
